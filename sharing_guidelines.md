# Safe Social Sharing Guidelines

These guidelines dictate how KaroTools links should be shared on social platforms like Reddit, Quora, Twitter, and LinkedIn.

## 1. No Automated Posting
We do not use bots, scripts, or automated agents to mass-post links to Reddit or Quora. All sharing must be done manually and thoughtfully.

## 2. No Spammy Backlinks
Do not drop links in unrelated threads. Do not create fake accounts to upvote your own links. Do not purchase backlinks or use PBNs (Private Blog Networks).

## 3. Genuine Value Only
Only share a link if it genuinely answers the user's specific question. 
When sharing, you must provide a useful, contextual explanation in the post itself so the user gets value even if they don't click the link.

## 4. Full Transparency
If you are the maintainer, disclose it clearly (e.g., "Full disclosure: I built this free tool to help with this exact calculation."). Do not pretend to be an unbiased third party discovering the site.

## 5. Accurate Previews
Ensure all Open Graph (OG) tags and Twitter cards accurately reflect the specific page being shared. Do not use clickbait titles or misleading descriptions to bypass moderation or trick users into clicking.
